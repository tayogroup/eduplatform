# Illustration prompt set

Lulu journey series (Term 3). The current page-NN.svg files are original
vector illustrations; for painted pages use the image-generation tool in
illustration-story mode with the shared spec below. Per-page prompts: use
the page alt texts in shared/course-ui.js plus STORY.txt.

## Shared visual specification

- Original hand-painted Grade 1 picture-book illustration; soft gouache
  and colored-pencil texture with subtle paper grain.
- Same storyworld as the Musa, Kiki and Duku books.
- Lulu is a little swallow: deep blue back and wings, cream chest, rust
  throat, forked tail, big bright eyes.
- Journey dressing: winding river, the great blue lake, a small sailboat
  with white and coral sails, a big-leaf rain shelter, and a cheerful
  city of colorful buildings, a striped market stall, lamp posts and a
  clock tower; windows and lamps glow gold at night.
- No text, logos, frames, watermarks, humans, scary expressions, or
  duplicated limbs.
