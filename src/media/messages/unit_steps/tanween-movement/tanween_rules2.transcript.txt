[[section:hero]]
[thoughtful] Today, we are learning Tanween. [pause] Tanween is the sound of an extra "n" added to the end of a word. [pause] Tanween is written using two Harakah marks. [pause] There are three types of Tanween: Tanween Fatḥ, Tanween Kasr, and Tanween Ḍamm. [pause] Let's learn them together.

[[section:rule-1]]
[excited] Rule Number One!

[pause]

Tanween makes the "n" sound. Listen carefully:

[pause]

كِتَابٌ — Kitābun.

[pause]

كِتَابٍ — Kitābin.

[pause]

كِتَابًا — Kitāban.

[pause]

Notice the light "n" sound at the end of each word.

[[section:rule-2]]
[excited] Rule Number Two!

[pause]

There are three types of Tanween. Tanween Fatḥ makes the sound "an," as in:

[pause]

كِتَابًا — Kitāban.

[pause]

Tanween Kasr makes the sound "in," as in:

[pause]

كِتَابٍ — Kitābin.

[pause]

Tanween Ḍamm makes the sound "un," as in:

[pause]

كِتَابٌ — Kitābun.

[[section:rule-3]]
[excited] Rule Number Three!

[pause]

Read the two Harakah marks together because they make one sound. Listen:

[pause]

بٌ — Bun.

[pause]

بٍ — Bin.

[pause]

بً — Ban.

[pause]

Do not read the two marks separately.

[[section:rule-4]]
[excited] Rule Number Four!

[pause]

Tanween is always read at the end of a word. For example:

[pause]

مُسْلِمٌ — Muslimun.

[pause]

رَحِيمٍ — Raḥīmin.

[pause]

عَلِيمًا — ʿAlīman.

[pause]

Read the Tanween after the last letter.

[[section:rule-5]]
[excited] Rule Number Five!

[pause]

Tanween Fatḥ often has an extra Alif. For example:

[pause]

كِتَابًا — Kitāban.

[pause]

The Alif helps show the Tanween, but do not read the Alif separately.

[[section:rule-6]]
[excited] Rule Number Six!

[pause]

Some words do not have the extra Alif. For example:

[pause]

رَحْمَةً — Raḥmatan.

[pause]

شَيْئًا — Shay'an.

[pause]

Read the Tanween normally.

[[section:rule-7]]
[excited] Rule Number Seven!

[pause]

Read Tanween clearly. Listen:

[pause]

بٌ — Bun.

[pause]

بٍ — Bin.

[pause]

بً — Ban.

[pause]

Make the "n" sound easy to hear.

[[section:rule-8]]
[excited] Rule Number Eight!

[pause]

When stopping,

[pause]

the Tanween sound usually disappears. For example,

[pause]

كِتَابٌ is read as Kitābun

[pause]

when continuing,

[pause]

but Kitāb

[pause]

when stopping.

[pause]

كِتَابٍ is read as Kitābin

[pause]

when continuing,

[pause]

but Kitāb

[pause]

when stopping.

[pause]

كِتَابًا is read as Kitāban

[pause]

when continuing,

[pause]

but Kitābā

[pause]

when stopping.

[[section:rule-9]]
[excited] Rule Number Nine!

[pause]

Read Tanween smoothly and gently. Do not rush. Give every Tanween its proper sound and pronounce the "n" clearly.
