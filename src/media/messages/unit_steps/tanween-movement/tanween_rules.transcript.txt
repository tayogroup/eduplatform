
[thoughtful]
[[section:hero]]
Today, we are learning Tanween.

[pause]

Tanween is the sound of an extra "n" added to the end of a word.

[pause]

Tanween is written using two Harakah marks.

[pause]

There are three types of Tanween.

[pause]

Tanween Fatḥ.

[pause]

Tanween Kasr.

[pause]

And Tanween Ḍamm.

[pause]

Let's learn them together.


[excited]
[[section:rule-1]]
Rule Number One!

[pause]

Tanween makes the "n" sound.

[pause]

Listen carefully.

[pause]

كِتَابٌ

[pause]

Kitābun.

[pause]

كِتَابٍ

[pause]

Kitābin.

[pause]

كِتَابًا

[pause]

Kitāban.

[pause]

Notice the light "n" sound at the end.


[excited]
[[section:rule-2]]
Rule Number Two!

[pause]

There are three types of Tanween.

[pause]

Tanween Fatḥ makes the sound:

an.

[pause]

Example:

كِتَابًا

[pause]

Kitāban.

[pause]

Tanween Kasr makes the sound:

in.

[pause]

Example:

كِتَابٍ

[pause]

Kitābin.

[pause]

Tanween Ḍamm makes the sound:

un.

[pause]

Example:

كِتَابٌ

[pause]

Kitābun.


[excited]
[[section:rule-3]]
Rule Number Three!

[pause]

Read the two Harakah marks together.

[pause]

They make one sound.

[pause]

Listen:

بٌ

[pause]

Bun.

[pause]

بٍ

[pause]

Bin.

[pause]

بً

[pause]

Ban.

[pause]

Do not read the two marks separately.


[excited]
[[section:rule-4]]
Rule Number Four!

[pause]

Tanween is always read at the end of a word.

[pause]

For example:

مُسْلِمٌ

[pause]

Muslimun.

[pause]

رَحِيمٍ

[pause]

Raḥīmin.

[pause]

عَلِيمًا

[pause]

ʿAlīman.

[pause]

Read the Tanween after the last letter.


[excited]
[[section:rule-5]]
Rule Number Five!

[pause]

Tanween Fatḥ often has an extra Alif.

[pause]

For example:

كِتَابًا

[pause]

Kitāban.

[pause]

The Alif helps show the Tanween.

[pause]

Do not read the Alif separately.


[excited]
[[section:rule-6]]
Rule Number Six!

[pause]

Some words do not have the extra Alif.

[pause]

For example:

رَحْمَةً

[pause]

Raḥmatan.

[pause]

شَيْئًا

[pause]

Shay'an.

[pause]

Read the Tanween normally.


[excited]
[[section:rule-7]]
Rule Number Seven!

[pause]

Read Tanween clearly.

[pause]

Listen:

بٌ

Bun.

[pause]

بٍ

Bin.

[pause]

بً

Ban.

[pause]

Make the "n" sound easy to hear.


[excited]
[[section:rule-8]]
Rule Number Eight!

[pause]

When stopping, the Tanween sound usually disappears.

[pause]

For example:

كِتَابٌ

[pause]

When continuing:

Kitābun.

[pause]

When stopping:

Kitāb.

[pause]

كِتَابٍ

[pause]

When continuing:

Kitābin.

[pause]

When stopping:

Kitāb.

[pause]

كِتَابًا

[pause]

When continuing:

Kitāban.

[pause]

When stopping:

Kitābā.


[excited]
[[section:rule-9]]
Rule Number Nine!

[pause]

Read Tanween smoothly and gently.

[pause]

Do not rush.

[pause]

Give every Tanween its proper sound.

[pause]

Pronounce the "n" clearly.
