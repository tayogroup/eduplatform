# The Big Friends Party - illustration prompt set

Capstone crossover. The current page-NN.svg files are original vector
illustrations; for painted pages use the image-generation tool in
illustration-story mode with the shared specs of all four series
(see the PROMPTS.md files of musas-muddy-stripes, kiki-goes-to-school,
duku-makes-a-scarecrow and lulu-says-lets-go). Per-page prompts: use
the page alt texts in shared/course-ui.js plus STORY.txt.

Every character keeps their established design. Party dressing: the
bunting banner from Musas Big Race, the long bench table, and the
rainbow finale shared by every series ending.
