# Pre-Quraan App

Development repository for Pre-Quraan learning units.

## Golden Unit

- Alphabet

## Development

```bash
npm install
npm run dev
```

## Runtime Bundles

Edit unit runtime source in `src/units/<unit-key>/runtime/*.js`. The browser bundle `runtime.bundle.js` is generated; see `docs/generated-bundle-policy.md`.

## Naming And Versioning

Active JavaScript and CSS filenames are stable and do not include versions. Versions live in Git tags, release notes, and manifests; see `docs/naming-versioning.md`.
